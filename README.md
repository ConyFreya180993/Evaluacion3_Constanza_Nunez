##Descripción
Este proyecto corresponde a la Evaluación 3 de la asignatura Programación Web.  
Incluye dos ejercicios implementados con **Flask**:

1. **Ejercicio 1:** Formulario de notas y asistencia.  
   - Calcula el promedio de tres notas.  
   - Verifica asistencia mínima.  
   - Muestra estado: Aprobado/Reprobado.

2. **Ejercicio 2:** Formulario de nombres.  
   - Recibe tres nombres.  
   - Devuelve el nombre más largo y su cantidad de caracteres.

- Python 3.14

##Ejecución
1. Clonar este repositorio:
```
   git clone https://github.com/ConyFreya180993/Evaluacion3_Constanza_Nunez.git
```

2. Entrar a la carpeta del proyecto
```bash
cd Evaluacion3_Constanza_Nunez 
```
3. Ejecutar servidor Flask
```
python main.py
```
4. Abrir en navegador
```  
http://127.0.0.1:5000
```
